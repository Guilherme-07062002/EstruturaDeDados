# Bubble Sort (Pseudocódigo)

```pseudocodigo
função bubbleSort(vetor)
inicio
    n <- tamanho do vetor
    para i de 0 ate n faça
        para j de 0 ate n - 1 faça
            se vetor[j] < vetor[j + 1] então
                trocar(lista[j], lista[j + 1])
            fim_se
        fim_para
    fim_para
fim
```
